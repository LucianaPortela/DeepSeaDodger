 ![alt text](https://github.com/LucianaPortela/DeepSeaDodger/blob/master/asset/PlayerBg0.png?raw=true "Deep Sea Dodge") Deep Sea Dodge 🌊

Deep Sea Dodge é um jogo 2D de sobrevivência subaquática desenvolvido em Python utilizando a biblioteca Pygame. O objetivo é guiar seu peixinho através das profundezas, desviando de predadores perigosos e coletando minhocas para recuperar vida e pontuar.
